System Summary

SORT v2 (Smart Operational Recovery and Tracking System) is a comprehensive Material Recovery Facility management system with the following verified features:

🎯 Core Workflows

  1. Report Submission: Students/Teachers submit waste or asset reports with location, photos, and notes
  2. Admin Validation: Admin reviews and verifies or dismisses reports
  3. Staff Dispatch: Admin assigns verified reports to MRF staff members
  4. Collection Tracking: MRF staff logs material weight and confirms collection
  5. Task Completion: MRF staff marks tasks as completed, triggering point awards
  6. Gamification: Reporter receives 20 points upon report completion
  7. Communication: Admin publishes campus news and updates

👥 User Roles

  • STUDENT: Submit reports, view points, view leaderboard
  • TEACHER: Submit reports, view points
  • ADMIN: Validate reports, dispatch staff, publish news, view analytics
  • MRF: View assigned tasks, log collection weight, mark completed

📊 Database Structure

  • User: Profile, role, points, reports count, gamification data
  • Report: Location, type (WASTE/ASSET), status, assigned staff, kilos collected, collection date
  • CampusNews: Announcements with tag, date, title, description

🔄 Status Lifecycle

PENDING → VERIFIED (admin approval) → DISPATCHED (staff assigned) → IN_PROGRESS (collecting) → COMPLETED (done, points awarded)
Alternative: PENDING → DISMISSED (invalid/duplicate)
